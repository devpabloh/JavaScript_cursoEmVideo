Operadores aritméticos

Os operadores aritméticos em javaScript são:

  • + soma
  • - subtração
  • * multiplicação
  • / divisão
  • % resto da divisão
  • ** potência

Ordem de precedência

Observação: Devemos lembrar que o JavaScript leva em conta a ordem de precedência, assim como na matemática, ou seja, se no mesmo cálculo tiver uma soma e depois uma divisão, o cálculo da divisão vai ser realizado primeiro, para depois ser realizado a soma. Mas para que a gente faça a soma primeiro que a divisão podemos colocar ela entre parênteses () porque primeiro é realizado o cálculo que está dentro dos parênteses

  1. parênteses
  2. potência
  3. multiplicação, divisão, resto da divisão
  4. soma, subtração

Observação: Quando você coloca vários sinais de mesmo nível de precedência no mesmo cálculo, ele vai fazer os calculos de acordo com essa lista, só que começando pelo primeiro sinal até o último, da esquerda para direita, o sinal que aparecer primeiro.
